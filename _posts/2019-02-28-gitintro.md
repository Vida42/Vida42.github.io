### [git<Commit和Push的区别>](https://www.cnblogs.com/runningdonkey/p/6932407.html)

git作为支持分布式版本管理的工具，它管理的库（repository）分为本地库、远程库。
git commit操作的是本地库，git push操作的是远程库。

git commit是将本地修改过的文件提交到本地库中。
git push是将本地库中的最新信息发送给远程库。
